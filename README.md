# Qraw

A fast, privacy-focused QR code generator. No accounts, no tracking, no data collection.

Built with React, TypeScript, and a soft Neo-Brutalist design.

## Features

- **Website URLs** - Generate QR codes for any link
- **Wi-Fi Networks** - Share network credentials instantly
- **Social Media** - Quick profile links with embedded logos
- **Custom Titles** - Add headings to your QR codes
- **Background Images** - Personalize with custom backgrounds
- **Multiple Formats** - Export as PNG, SVG, or PDF

## Why Qraw?

Most QR generators track your data or require accounts. Qraw runs entirely in your browser. Your data never leaves your device.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- qrcode library

## Privacy

- No backend servers
- No analytics or tracking
- No cookies
- All processing happens locally
- Works offline after first load
